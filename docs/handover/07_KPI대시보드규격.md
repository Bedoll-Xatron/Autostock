## 2. 핵심 KPI (Critical)

### 2.1 일일 손절률 (Daily Stop-Loss Rate)

**정의:**
```
손절률 = (당일 손절 매도 건수) / (당일 보유 종목 수) × 100
```

**산출:**
```sql
WITH today_buys AS (
  SELECT ticker, name FROM decisions
  WHERE DATE(created_at) = CURRENT_DATE
    AND action = 'BUY' AND order_qty > 0
),
today_sells AS (
  SELECT ticker, reason FROM held_position_history  -- 또는 sells 테이블
  WHERE DATE(sold_at) = CURRENT_DATE
    AND reason IN ('손절', '손절(갭다운)')
)
SELECT
  COUNT(DISTINCT s.ticker)::FLOAT / GREATEST(COUNT(DISTINCT b.ticker), 1) * 100 AS stop_rate
FROM today_buys b
LEFT JOIN today_sells s ON s.ticker = b.ticker;
```

**목표 추이:**
- W0 (현재): 추정 50~70% (사용자 호소)
- W1 후: 30~40% 기대
- W6 후: 20% 이하

---

### 2.2 평균 진입 슬리피지 (Average Entry Slippage)

**정의:**
```
slippage_pct = (체결가 - 전일종가) / 전일종가 × 100
```

**해석:**
- +값: 갭업 자리에서 매수 (위험)
- 0 부근: 어제 종가 부근 매수 (이상적)
- -값: 갭다운 매수 (가치 진입 또는 추세 훼손 둘 중 하나)

**저장 위치:** `decisions.entry_slippage_pct` (W1에서 추가)

---

### 2.3 평균 손익 R (Average R-multiple)

**정의:**
```
R = 실현 손익 / 진입 시 위험 금액
  = (매도가 - 진입가) × qty / ((진입가 - stop_price) × qty)
  = (매도가 - 진입가) / (진입가 - stop_price)
```

**해석:**
- 손절 시 R = -1
- 익절 +5%, 손절폭 3% 시 R = +1.67
- 평균 R > 0이면 시스템이 양의 기댓값

**목표:**
- W0: -0.3 ~ -0.5 추정
- W6 후: +0.2 이상

---

## 3. 보조 KPI (Supplementary)

### 3.1 게이트 차단률 (W4부터)

```
차단률 = 차단 건수 / 분석 종목 수 × 100
```

너무 높으면(>70%) 가드 과보수, 너무 낮으면(<5%) 가드 무용지물.

### 3.2 종목별 stop_pct 분포 (W2부터)

ATR 기반 손절폭이 시장 변동성을 반영하는지 검증.
- 예상 분포: 2.5% (FLOOR) ~ 6.0% (CEIL) 사이 골고루
- 모두 6.0%에 몰리면 변동성 큰 종목만 매수하는 신호

### 3.3 갭다운 회복 성공률 (W3부터)

```
회복률 = (5분 대기 후 stop 이상 복귀) / (갭다운 감지 건수) × 100
```

### 3.4 섹터 분포 (W5부터)

동시 보유 종목의 섹터 다양성. Herfindahl-Hirschman 지수 또는 단순 sector_count 분포.
