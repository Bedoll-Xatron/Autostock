## 0. 한 줄 요약

> "실제 수익(알파)을 내라"는 요청으로 전략을 정밀 백테스트한 결과, **가격/패턴/수급/VCP에는 선별 알파가 없고(수익=강세장 베타), 오직 '6개월 모멘텀'에만 실제 횡단면 알파(+1.98%p)가 있음**을 확인. 이에 (1) 청산을 손익비 우선으로 재설계, (2) 사이징을 ATR 단독화, (3) **scorer 상대강도를 20일→6개월 모멘텀으로 교체**하여 배포. 현재 관찰 중.

## 2. 연구 결과 (백테스트, 중소형 KOSPI+KOSDAQ, ~1년)

| 검증 | 결과 | 결론 |
|---|---|---|
| 가격 진입(대형/중소형) | alpha −1.0~−2.0%p | 엣지 없음 |
| 눌림 vs 강세추종 | 눌림이 손익비 우위, 단 alpha ≤0 | 타이밍만 개선 |
| 수급 외국인(대형) | IC ≈ 0 | 엣지 없음 |
| 수급 외국인(중소형) | IC +0.10, 스프레드 +4.51%p | 단독 +, **결합 시 반전** |
| **VCP 실제 전략** | alpha −1.29%p, **A등급 −5.84%p(최악)** | 플래그십도 엣지 없음 |
| 단기(20일) 모멘텀 | IC 음수 | 단기는 평균회귀 |
| **6개월 모멘텀** | **IC +0.146, 상위20% alpha +4.94%p, 날짜중립 L/S +12.92%p** | ✅ **실제 알파** |
| Step D 결합 | 모멘텀 상위 단독 alpha **+1.98%p**, 기댓값 +7.13%, 손익비 3.18, 승률 46.4% | ✅ 결합에도 생존(외국인과 달리) |

**핵심:** 시스템은 **단기 강세추종**(틀린 horizon)으로 사고 있었음. 정석 모멘텀(3~6개월, Jegadeesh-Titman)이 한국 중소형주에서도 작동. **눌림은 모멘텀을 희석**하므로 모멘텀 선정엔 불필요.
